DIFFERENCE BETWEEN PADDINGS, MARGINS AND BORDERS

MARGINGS PADDINGS BORDERS
Margin is the outermost layer or the space outside the border Padding is the space between the content of an element and its border Border refers to the container between the padding and margin or the box wrapping the element.
In the css box-model, margins are not added in the calculation of the width and height of an element In the css box-model, paddings are added in the calculation of the width and height of an element In the css box-model, borders are added in the calculation of the width and height of an element
